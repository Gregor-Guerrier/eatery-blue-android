# Eatery Blue - Browse. Search. Eat.

<p align="center"><img src=https://raw.githubusercontent.com/cuappdev/eatery-blue-android/main/eatery-blue-banner.png width=500 /></p>

Eatery was the first app released by AppDev.  With over 6,000 students using it every month, it enables students to browse menus and discover places to eat on Cornellâ€™s campus. It enhances the dining experience at Cornell with features such as providing the crowdedness of eateries, checking meal swipes and dining money balances, and favoriting dishes. Eatery is available on both [iOS](https://github.com/cuappdev/eatery-blue-ios) and Android platforms.

## Installation
Clone this repository and import into **Android Studio**
```bash
git clone git@github.com:cuappdev/eatery-blue-android.git
```

## Generating signed APK
From Android Studio:
1. ***Build*** menu
2. ***Generate Signed APK...***
3. Fill in the keystore information *(you only need to do this once manually and then let Android Studio remember it)*
